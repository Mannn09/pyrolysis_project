# Rice Husk Pyrolysis Documentation

## Physics Model
- **Method:** Finite Volume Method (FVM)
- **Solver:** Penta-diagonal Implicit Matrix Solver
- **Coordinates:** 2D Cylindrical (R, Z)

## Parameters
- Effective Thermal Conductivity: 0.08 W/mK
- Bulk Density: 400 kg/m3

## Expected Results
This model calculates the temperature distribution and gas pressure 
evolution over 10 minutes of heating at 60°C/min.
